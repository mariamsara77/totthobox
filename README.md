# Totthobox

Totthobox হলো একটি মাল্টি-সার্ভিস ডিজিটাল পোর্টাল যা রিজিওনাল ডেটা, নিউজ হেডলাইন এবং শিক্ষামূলক রিসোর্স একত্রিত করে।

## 🌐 Live Project
সরাসরি সাইটটি ভিজিট করুন: **[https://totthobox.com](https://totthobox.com)**

## 🚀 Features
*   **Regional Data Aggregation:** বিভিন্ন অঞ্চলের তথ্যের সহজলভ্যতা।
*   **News Headlines:** প্রতিদিনের আপডেট নিউজ হেডলাইন।
*   **Educational Resources:** শিক্ষার্থীদের জন্য প্রয়োজনীয় শিক্ষামূলক রিসোর্স।

## 🛠 Tech Stack
এই প্রজেক্টটি আধুনিক ওয়েব প্রযুক্তিতে তৈরি:
*   **Framework:** Laravel 12
*   **Frontend:** Livewire 4, Volt Components, Flux UI 2
*   **Database:** MySQL
*   **Search & Cache:** Meilisearch & Redis
